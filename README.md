# rjgc_myglxt
## 需要的软件
- java
- jar1.8以上
- eclipse
- eclipse的spring boot 插件
- mysql
- mysql创建rjgc数据库
- 新的数据库文件已更新：在src/main/resources/db下
- 修改src/main/resources/application.properities文件里的数据库信息可自动连接并建表
- 若报数据库错误，先删除rjgc数据库，再新建即可。
